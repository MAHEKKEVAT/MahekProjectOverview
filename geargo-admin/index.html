📍 Overview

The Admin portal is a Flutter (Dart) web / desktop companion to the customer GearGo app. It gives verified staff a real-time view of cars, bookings, revenue, user analytics and damage reports. All data lives in Firebase; images are cached via Appwrite CDN for speed. Built specifically for GearGo car owners and managers to monitor their fleet performance, track rental history, manage vehicle maintenance, and maximize revenue streams efficiently.

⚙️ Core Admin Features

Feature Description
Auth Secure sign-in / sign-up with role-based access (Admin ID check)
Dashboard KPI cards – total cars, active bookings, revenue, customers at a glance
Fleet Add / edit / delete cars; upload 4 images; set price, subscription tier
Bookings Accept / reject / cancel; real-time status; live GPS tracking integration
Damage Image-based damage reports; approve cost, notify user automatically
Invoice Auto-generate PDF + e-mail; Razorpay ref & tax break-up included
Revenue City-wise, user-wise, car-wise analytics; weekly / monthly trend analysis
Analytics Bar & pie charts – bookings, user distribution, profit margin visualization
Customers CRUD users; view licence, booking history, total spend per customer
Reviews Moderate car reviews; hide / show based on quality standards
Profile Edit admin info; logout; exit app with confirmation dialogs
